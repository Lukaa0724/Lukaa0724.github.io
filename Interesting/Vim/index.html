    感觉自己英语打的速度好慢 借助vimtutor正好练习一下

    There are 7 lessons in the vimtutor.

    lesson 1

    remember that :q! is exit. There is a ":".

    append text(add something at the end of the line) The KET "A" NOTE: It's a big.

    SAVE and EXIT the vim :wq it's easy.

    lesson 1 has been finished. Then the next.

    lesson 2 deletion commands

    move the cursor to the beginning of the word ,ESC and dw to delete the word.

    NOTATION: if you type the 'd' but the wrong character for the next one ,using ESC to start over.

    What's more? I find something interesting. I type the "dd" and 'dj' wrong, and the all line is deleted.

    d$ to delete to the end of the line.

    operators and motions : d means the delete operator. motion is what the operator will operator on.

    dw will delete until the next word(it will delete the space)

    de will delete only the word, will not delete the space.

    use count for a motion for example

    2w to move the cursor to the 2 words forward.

    3e to move the cursor to the 3 words forward, but the cursor will move to the end of the word.

    0 to move the cursor to the start of the line.

    using a count to delete more

    its format : d number motion (for example d2w, which means delete the 2 words )

    operatting on lines

    dd to delete the whole line. And 2dd to delete the two lines.

    the undo command

    it equals ctrl+z in windows (u)

    the next state(equals to ctrl+y in windows) ctrl + R

    and use U to return the line to its original state.

    Lesson 3.1 The put command

    p to put the text deleted previously after the cursor.

    the replace command

    r refers to replace.

    r+ the character you want to change.(for example ra) replace the character at the cursor with a

    the change operator

    ce means delete the word and places you in Insert mode .

    more changes using c

    c [number] motion

    number is not necessary.

    c$ means to delete the all words after the cursor and change to insert mode,

    Lesson 4.1 Cursor location and file status

    ctrl + g tells you your location in the file.

    And type G to move to a line in the file.

    G to move the cursor to the bottom of the file.gg to move the cursor to the start of file.

    And it's usual to firstly CTRL+G to show the location, and I can type line + G to jump to the line.(556G to move to the line 556)

    the search command

    type / to search some word. And it searches from the beginning to the end.

    type ? to search some word too. But it searches from the end to the beginning.

    And I can type CTRL + o to come back the cursor I press / or ?

    matching parentheses search

    parentheses n.括号 such as there is a (, but I cannnot find the next ), I can move the cursor to the ( and type the ) to find the next ).

    This is very useful to debug.

    the substitute command

    s means substitute.

    for exampleL: s/old/new/g subtitute new for old 寻找old 改为new

    another example: :s/thee/the find the thee and change them to the.

    There are some differences. :s/thee/the means to change only one thee to the, however, :s/thee/the/g means to change all line globally thee to the.

    NOTE : must type :::::::::::::::::::::::::::::::::::::::

    Actually, we will substitute words not only one line, so we can use the command :555,557s/old/new/g means to change line 555--557 old for new.

    :%s/old/new/g means change in the whole file,

    :%s/old/new/gc means change and judge whether change or not,

    Lesson 5.1 how to execute an external command

    exclamation n感叹号 ！

    :! + Linux shell such as :!ls(It will jump to the shell and tell you the content)

    more on writing files

    save the file :w+FILENAME( :w TEST which means save the file and name it TEST) and I can use external command :!rm TEST to remove the file TEST.

    Selecting text to write

    press "v" to start the Visual selection. And now the cursor will select some words and highlighted them. I can take many actions about it. For example, after typing v and select some words, type :w TEST, which will save the file TEST.

    retrieving and merging files(检索和合并文件)

    retrieve v.检索 找回

    merge v.融入 合并

    to insert the contents of a file, type :r FILENAME

    a complex example: :r !ls retrieve the content of 'ls' to the file.

    Lesson 6.1 the open command

    o and O o means jump to next line and start the insert mode. O means jump to last line and start the insert mode.

    the append command

    type a to insert text after the cursor.

    Now I have learned 3 insert modes about the vim. - a : insert after the cursor - i : insert at the cursor - A : append at the end of the line

    They are different, but I think A and a are very useful.

    Another way to replace

    Type R to replace more than one character,.

    Let's recall the use of r. It can only remove one character and replace it with one character. But R will replace the words until you type ESC.

    copy and paste text

    use y operator to copy text and p to paste it.

    First I must get into Visual mode. And we must type v to get into Visual mode, and I will select some words and then I type y(yank) (also called copy) the selected words and then type p to paste what I have copied.

    TIPS: I can use yw to copy a word.

    set option

    set a option so a search or substitute ignores case.

    if I want to search something and ignore the case(忽略大小写)

    /someword first find the word, then :set ic standing for (ignore case)(忽略大小写). And I can type :set hls to highlight the word searched. To remove the highlighting of the matched words, I can type :nohlsearch. To end the ignore case, type :set noic.

    such are something important in vimtutor: Typing ":set xxx" sets the option "xxx". Some options are:

        - 'ic' 'ignorecase'       ignore upper/lower case when searching
    - 'is' 'incsearch'        show partial matches for a search phrase
    - 'hls' 'hlsearch'        highlight all matching phrases
